# 🧠 Вікторина (Python Tkinter)

Простий інтерактивний застосунок-вікторина, розроблений на Python з використанням бібліотеки Tkinter.

---

## 📌 Опис проєкту

Вікторина — це графічний застосунок, який дозволяє користувачу проходити вікторину з кількома питаннями та отримувати результат у вигляді балів.

---

## ⚙️ Функціонал

- відображення питань по черзі
- 4 варіанти відповіді
- перевірка правильності відповіді
- підрахунок балів
- відображення результату після завершення гри
- можливість перезапуску вікторини

---

## 🚀 Запуск проєкту

### 1. Клонування репозиторію
```bash
git clone https://github.com/Arzenh/QAprKholoshevskyi.git
cd QAprKholoshevskyi
```
Запуск програми

python Kholoshevskyipr1.py

## 🛠 Технології

- Python 3.14.4
- Tkinter (GUI)

## 🧩 Як працює застосунок

1. Користувач запускає програму
2. Відкривається вікно "Вікторина"
3. Відображається питання та 4 варіанти відповідей
4. Користувач обирає відповідь кнопкою
5. Програма показує:
   - ✔ правильна відповідь
   - ❌ неправильна відповідь
6. В кінці показується фінальний результат
